# Weather Cast

A weather application built using HTML, CSS and JavaScript.

## Features

- Search weather by city
- Live weather data from OpenWeatherMap API
- Async/Await Fetch API
- Weather icons
- Error handling
- Geolocation support
- LocalStorage caching
- Dynamic background changes
- Enter key search

## Technologies

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
